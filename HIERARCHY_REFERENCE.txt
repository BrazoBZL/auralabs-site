SPACING AS AUTHORITY
================================================================================

V1 SPACING:
┌──────────────────────────┐
│ Hero    | 80px padding   │ (60vh height)
├──────────────────────────┤
│ Why Us  | 80px padding   │ (three columns, separated)
├──────────────────────────┤
│ Products| 80px padding   │ (6 cards, inventory feel)
├──────────────────────────┤
│ Process | 80px padding   │ (4 steps, operational)
├──────────────────────────┤
│ Footer  | 64px padding   │
└──────────────────────────┘
  Result: Uniform spacing = equal importance = competing messages

V2 SPACING:
┌──────────────────────────────┐
│ HERO    | 120px padding      │ ← EMPHASIS (85vh height)
│         | (massive whitespace)│ ← BELIEF DOMINATES
├──────────────────────────────┤
│ Position| 96px padding       │ ← SUPPORT (white bg)
│         | (institutional voice)│ ← EXPLAINS WHY
├──────────────────────────────┤
│ Products| 96px padding       │ ← EVIDENCE (curated)
│         | (48px card gaps)    │ ← BREATHING ROOM
├──────────────────────────────┤
│ Process | 80px padding       │ ← ACCESS (smaller)
│         | (3 steps, tight)    │ ← SUBORDINATE
├──────────────────────────────┤
│ Footer  | 64px padding       │
└──────────────────────────────┘
  Result: Graduated spacing = clear hierarchy = single narrative

EQUATION:
  More padding = More authority
  More breathing = More selectivity
  Reduction = Curation

================================================================================
THE 10-FOOT TEST
================================================================================

V1 FROM 10 FEET AWAY:
  ✓ Can read multiple headings (Aura Labs / We offer / How We Maintain / 
    Our Compounds / Getting Started)
  ✗ No single dominant message
  ✗ Reader asks: "What's the main thing here?"

V2 FROM 10 FEET AWAY:
  ✓ Can ONLY read: "We offer what we're certain of. Nothing more."
  ✓ Everything else: unreadable, just whitespace/shapes
  ✓ Reader immediately understands: THIS IS THE POINT

FROM 5 FEET AWAY:

V1 becomes clearer:
  Brands → Beliefs → Why → Products → Process
  Multiple narratives compete for attention

V2 becomes clearer:
  Belief → Why it matters → What made the cut → How to proceed
  Single narrative unfolds, each section supports the one before

HIERARCHY CHECKLIST
================================================================================

✅ V2 PASSES ALL CRITERIA:

[✓] Does the belief dominate?
    Hero: 85vh, 120px padding, single statement
    Result: Visually unavoidable, reads as institutional identity

[✓] Does the rest feel like evidence, not explanation?
    Position: Single institutional voice (not three columns)
    Products: "Selected" with 3 visible (curation, not inventory)
    Process: Gatekeeping language (qualification, not onboarding)

[✓] Does "selection" communicate better than "inventory"?
    V1 title: "Our Compounds" (here's what we have)
    V2 title: "Selected Compounds" (here's what we chose)
    Psychology: Selectivity = higher standards

[✓] Would a CEO read this and think "curated" not "optimized"?
    V1: Optimized presentation (multiple messages organized)
    V2: Curated selection (belief > position > evidence > access)
    Language shift: "Compounds" → "Selection" → "Intake"

[✓] From 10 feet away, is the hero the ONLY readable thing?
    V1: Multiple headings readable (5 competing messages)
    V2: Only hero readable (single belief dominates)
    Result: Clear hierarchy, no competing narratives
